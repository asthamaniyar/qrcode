### Command Quick Reference

**Installation:**
```bash
cd backend && npm install
cd ../frontend && npm install
```

**Running:**
```bash
# Option 1: Automated (Windows)
start.bat

# Option 2: Manual
# Terminal 1
cd backend && npm run dev

# Terminal 2
cd frontend && npm run dev
```

**Building for Production:**
```bash
cd frontend
npm run build
```
